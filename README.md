## Install

```sh
yarn add facelift
```

## Usage

```
Usage

   $ facelift <command> <params>

   $ facelift sample <param>             # Uses the <PARAM>
   
 Examples

   $ facelift sample TEST                # Uses the TEST
   $ facelift sample YOLO                # Uses the YOLO
```
